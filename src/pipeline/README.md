# Pipeline Source

This folder contains the GNN-relevant source code copied from the project:

- pseudo-label seed construction and inspection,
- spatial graph construction,
- feature and fold preparation,
- EdgeGated GraphSAGE training and inference,
- cross-region validation.

The scripts preserve the original cluster paths used during development. See
[docs/reproduction_notes.md](../../docs/reproduction_notes.md) before rerunning,
and [docs/reuse_guide.md](../../docs/reuse_guide.md) before adapting the code
outside the original cluster project.
